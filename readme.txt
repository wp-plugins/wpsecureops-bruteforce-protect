== Description ==

The simplest yet free Brute Force plugin that will protect your WordPress site from any brute force password attacks.

No coding skills required!

Automatically protect your site from brute force password attacks to /wp-admin/ or any login form generated by WordPress

== Installation ==

1. Install via wordpress.org
1. Activate the plugin through the 'Plugins' menu in WordPress
1. _(optional)_ Go to the plugin's settings page and alter (if you need) the brute force parameters.
